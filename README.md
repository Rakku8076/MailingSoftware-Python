# MailingSoftware-Python
Mailing Software is the software that deals with the issues related to the sending of an email. This project helps the user to send an email through Gmail account.
It provides them with user friendly interface and performs the following operations:
1.Sending email to a single receiver
2.Sending email to multiple receivers
3.Sending email with attachments
4.Sending email multiple times after login and before logout.

Features:-
1.It enhances the communication channel within the Gmail user accounts.
2.User can easily send text messages and multiple attachments in one mail.
3.User can also send email to multiple receiver with text messages and multiple attachments.
4.Mailing Software provides with the login and logout features and allow the user to send email multiple times.
5.This software can be used within an organization as it is reliable, cost-effective and efficient.
